# SearchCulture
---
Proyecto en la nube que será capaz de recomendar contenido (Películas, Series, Libros y Música) a los usuarios, en base a sus gustos (o a los gustos que deseen encontrar). El proyecto te hará una serie de preguntas con el fin de procesarlas para conocer un poco la personalidad adecuada, y dependiendo de las respuestas, generar unos resultados acordes. En resumidas cuentas, es un ___buscador de contenido según gustos y personalidad.___
